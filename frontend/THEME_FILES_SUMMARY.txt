🚀 DÉMARRAGE RAPIDE
═══════════════════════════════════════════════════════════════════

Option 1 - Tester la page de démo:
  └─ Ouvrez src/App.js
  └─ Importez: import ThemeDemo from './pages/ThemeDemo';
  └─ Affichez: return <ThemeDemo />;

Option 2 - Ajouter le switcher à votre Layout:
  └─ Ouvrez src/components/LayoutImproved.js
  └─ Importez: import ThemeSwitcher from './ThemeSwitcher';
  └─ Ajoutez: <ThemeSwitcher />

Option 3 - Utiliser le hook dans vos composants:
  import { useTheme } from 'next-themes';
  const { theme, setTheme } = useTheme();
